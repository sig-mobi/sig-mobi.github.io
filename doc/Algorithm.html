WebCrypto GOST Class: Algorithm

Algorithm

Constructor

new Algorithm(name)

The Algorithm object is a dictionary object [WebIDL] which is used to specify an algorithm and any additional parameters required to fully specify the desired operation.
 dictionary Algorithm {
     DOMString name;
 };
WebCrypto API reference http://www.w3.org/TR/WebCryptoAPI/#algorithm-dictionary
Parameters:
Name Type Description
name DOMString The name of the registered algorithm to use.
